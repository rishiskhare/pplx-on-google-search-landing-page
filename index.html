Frequently Asked Questions

Got questions? We’ve got answers.

Which browsers are supported?

Currently, the extension works on Chrome and all Chromium-based browsers such as Brave, Edge, and Opera.

Is Perplexity on Google Search really free?

Yes! The extension is completely free and open-source. There are no hidden costs or premium tiers.

Do I need an account to use the extension?

No account or sign-up is required. Just install the extension and start searching.

How is my data handled?

Your search queries never leave your browser. The extension does not collect or store any personal data.

How can I contribute to the project?

We welcome contributions! Visit the project’s GitHub repository to report issues, suggest features, or submit pull requests.

Is this extension created by Perplexity?

No. This project is an independent open-source effort and has no affiliation with Perplexity.
